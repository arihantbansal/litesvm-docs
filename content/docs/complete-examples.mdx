Explore complete, production-ready examples that demonstrate how to test Solana programs using LiteSVM with different frameworks.

## Framework Examples

### Anchor Framework

<Cards>
<Card
  title="Escrow Program"
  description="Token escrow implementation with Anchor"
  href="https://github.com/LiteSVM/litesvm-anchor-escrow-example"
/>
</Cards>

**Features:**
- Initialize escrow with token deposits
- Exchange tokens between parties
- Cancel and refund functionality
- PDA (Program Derived Address) management
- CPI (Cross-Program Invocation) testing

**Quick Start:**
```bash
git clone https://github.com/LiteSVM/litesvm-anchor-escrow-example
cd litesvm-anchor-escrow-example
cargo test
```

## What You'll Learn

From these complete examples, you'll understand:

- **Test Setup**: How to initialize LiteSVM for different program types
- **Account Management**: Creating and managing test accounts efficiently
- **Token Testing**: Working with SPL tokens in tests
- **Error Handling**: Testing error conditions and edge cases
- **Performance**: Measuring compute units and optimizing tests
- **Integration Testing**: Testing multiple programs together
- **State Verification**: Asserting program state changes correctly
