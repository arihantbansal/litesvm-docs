## Quick Method Reference

### Essential Methods

| Method | Description | Return Type |
|--------|-------------|-------------|
| `LiteSVM::new()` | Create a new test environment | `LiteSVM` |
| `send_transaction()` | Execute a transaction | `Result<TransactionMetadata, TransactionError>` |
| `get_account()` | Retrieve account data | `Option<Account>` |
| `airdrop()` | Fund an account with lamports | `Result<(), TransactionError>` |
| `add_program()` | Deploy a program from bytes | `Result<(), LiteSVMError>` |

### Configuration Methods

| Method | Description |
|--------|-------------|
| `with_compute_budget()` | Set compute budget limits |
| `with_sigverify()` | Enable/disable signature verification |
| `with_blockhash_check()` | Enable/disable blockhash validation |
| `with_lamports()` | Set initial account balance |

### Helper Functions

| Function | Description | Package |
|----------|-------------|---------|
| `CreateMint::new()` | Create a new token mint | `litesvm-token` |
| `CreateAssociatedTokenAccount` | Create an ATA | `litesvm-token` |
| `MintTo` | Mint tokens to an account | `litesvm-token` |
| `Transfer` | Transfer tokens between accounts | `litesvm-token` |

## Common Patterns

### Basic Test Setup

```rust
use litesvm::LiteSVM;
use solana_sdk::signature::{Keypair, Signer};

#[test]
fn test_my_program() {
    // Create test environment
    let mut svm = LiteSVM::new();

    // Deploy program
    let program_id = Pubkey::new_unique();
    let program_bytes = include_bytes!("../target/deploy/program.so");
    svm.add_program(program_id, program_bytes).unwrap();

    // Create and fund accounts
    let payer = Keypair::new();
    svm.airdrop(&payer.pubkey(), 10_000_000_000).unwrap();

    // Execute transactions...
}
```

### Error Handling

```rust
match svm.send_transaction(tx) {
    Ok(result) => {
        println!("Success! Compute units: {}", result.compute_units_consumed);
        println!("Logs: {:?}", result.logs);
    }
    Err(err) => {
        println!("Transaction failed: {:?}", err);
        // Access error details
        if let Some(logs) = err.meta.as_ref().map(|m| &m.logs) {
            println!("Transaction logs: {:?}", logs);
        }
    }
}
```
